Welcome to ngEHTforecast’s documentation!¶

ngEHTforecast is a set of tools written in Python for rapid forecasting of the science capabilities of the next-generation Event Horizon Telescope (ngEHT). These may be used to explore potential science cases and/or to shape the development and evolution of the ngEHT. ngEHTforecast is under continual development, so stay tuned!
